## [cite_start]PROJECT OVERVIEW [cite: 13]

[cite_start]The main objective is to develop a user-friendly system that verifies if an uploaded image or video frame is AI-generated[cite: 16]. [cite_start]This project helps users identify synthetic media by analyzing visual artifacts and metadata[cite: 17].

**Key Goals:**
* [cite_start]Reduce the spread of misinformation and online scams[cite: 17].
* [cite_start]Provide an accessible tool for academic and practical demonstration[cite: 27].
* [cite_start]Deliver fast processing times for end-users[cite: 48].

## [cite_start]SYSTEM FEATURES [cite: 42]

* [cite_start]**Media Verification:** Support for image uploads and single video frames[cite: 30, 43].
* [cite_start]**Detection Engine:** Lightweight deepfake detection using pre-trained models[cite: 32, 44].
* [cite_start]**Probability Score:** Displays a percentage score (e.g., 72%) indicating the likelihood of AI generation[cite: 34, 39, 45].
* [cite_start]**Metadata Analysis:** Inspects file metadata for traces of synthetic origin[cite: 38, 46].
* [cite_start]**User Interface:** A simple and clean UI for non-technical users[cite: 41, 47].

## [cite_start]METHODOLOGY [cite: 28]

The system follows a structured processing pipeline:
1. [cite_start]**Upload:** User provides the media file[cite: 30].
2. [cite_start]**Preprocessing:** The system performs resizing, normalization, and metadata extraction[cite: 31].
3. [cite_start]**Analysis:** Heuristic-based detectors look for texture inconsistencies and abnormal patterns[cite: 32, 33].
4. [cite_start]**Scoring:** The system calculates the probability of AI generation[cite: 34].
5. [cite_start]**Output:** Results are displayed in a clear format showing the likely source (Real or Synthetic)[cite: 35, 40].
